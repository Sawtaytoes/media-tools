# TagInputBase

Low-level primitive that combines a tag list with a text input. Used as the shared foundation for `LanguageCodesField` (autocomplete) and `FolderTagsField` (free-type).

## Props

| Prop | Type | Description |
|------|------|-------------|
| `tags` | `TagItem[]` | Tags to display. Each has `key`, `label` (ReactNode), and `title` (used as `aria-label` on the remove button). |
| `onRemove` | `(key: string) => void` | Called when a tag's remove button is clicked. |
| `inputProps` | `InputHTMLAttributes` | Spread onto the `<input>` element. Caller controls `value`, `onChange`, `onKeyDown`, `placeholder`, etc. |
| `children` | `ReactNode` | Rendered inside the `relative` wrapper below the input вҖ” use for autocomplete dropdowns. |
