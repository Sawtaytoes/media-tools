# VariablesSidebar

Persistent right-sidebar view of the sequence variables. Visible at `lg:` breakpoints and above; hidden on smaller screens where the Edit Variables modal is the only access point. Wraps `VariablesPanel` in an `<aside>` container.

## Responsive behavior

- **Below `lg`:** `hidden` — sidebar is not visible.
- **At `lg` and above:** `flex` — docked to the right of the sequence list.
