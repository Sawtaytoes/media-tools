# CommandHelpModal

A per-command reference modal showing the command summary, any note or output folder, and a full list of typed fields with descriptions.

Opens when the user clicks the ‚ďė button on a step card header. The `commandHelpModalCommandAtom` drives which command is shown; setting it to `null` (or `isOpen={false}`) hides the modal.

Field descriptions come from `window.getCommandFieldDescription` (populated from `src/api/schemas.ts` at prebuild).
