# LookupModal

Multi-stage search modal for resolving an external ID (MAL, AniDB, TVDB, TMDB, or DVDCompare) and writing it back into a step's field. The shape of `LookupState` (`stage`, `selectedGroup`, `selectedFid`, `selectedVariant`, `releases`, ...) is what drives stage transitions — the modal is a state-machine renderer.

The simple lookup types (MAL/AniDB/TVDB/TMDB) are single-stage: search → pick → close. **DVDCompare** is three-stage: search → pick a film → pick a variant (Blu-ray 4K / Blu-ray / DVD) → pick a specific release.

## Behaviour

- **Open** → `lookupModalAtom` is set with `{ lookupType, stepId, fieldName, stage: "search", … }`. The search input gets focused on the next tick.
- **Search** → Enter triggers `runSearch`. Posts `{ searchTerm }` to the type-specific endpoint and writes results back into the atom.
- **DVDCompare format filter** → `formatFilter` ("Blu-ray 4K" / "Blu-ray" / "DVD" / "all") narrows the result list pre-render.
- **Pick (simple types)** → calls `window.mediaTools.applyLookupSelection(stepId, fieldName, id, displayName)` and closes.
- **Pick (DVDCompare)** → if the group has 1 variant, auto-advances to the **release** stage. Otherwise transitions to **variant** stage. After variant pick, fetches releases; if there's exactly 1 it auto-selects and closes.
- **Back button** → only rendered in `variant` or `release` stage.
- **Keyboard** → `Esc` closes; `1-9` clicks the nth visible option button (only buttons with `data-option-index` are eligible). Inputs/textareas are exempt so typing doesn't trigger shortcuts.
- **Backdrop click** → close if the click target is the backdrop itself (not bubbled from inside).

## Atom wiring

| Atom | Role |
|------|------|
| `lookupModalAtom` | The full `LookupState` object: lookup type, target step/field, stage, search input, results, current selection (group/variant/release), error/loading flags |

## Server endpoints

| Endpoint | Purpose |
|----------|---------|
| `POST /queries/searchMal` | MAL anime search |
| `POST /queries/searchAnidb` | AniDB anime search |
| `POST /queries/searchTvdb` | TVDB series search |
| `POST /queries/searchMovieDb` | TMDB movie search |
| `POST /queries/searchDvdCompare` | DVDCompare film search (returns groups with variants) |
| `POST /queries/listDvdCompareReleases` | Releases for a chosen variant (`fid`) |

## Window globals consumed

| Global | Role |
|--------|------|
| `window.mediaTools.applyLookupSelection` | Writes a simple lookup result back into step state |
| `window.mediaTools.applyDvdCompareSelection` | Writes the full DVDCompare selection (group + variant + release hash) |
