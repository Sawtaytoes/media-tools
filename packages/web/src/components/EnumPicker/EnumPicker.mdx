# EnumPicker

A portal-rendered option list for enum-typed step fields (e.g., `episodeType` → regular / specials / credits / …). Pulls available options from the field's schema in `commandsAtom[stepCommand].fields[fieldName].options`.

Selecting an option dispatches `setParamAtom` to persist the value. Renders nothing when `enumPickerStateAtom` is `null`.

## Behaviour

- **Open** → `enumPickerStateAtom` is set with `{ anchor: { stepId, fieldName }, triggerRect }`. Initial highlight uses the step's current param value, or the field's `default` if unset.
- **Position** → 300 × 400 popover, left-aligned to the trigger. Flips above when there's < 200px below and more space up. Clamped to viewport with an 8px margin.
- **Search** → matches `label` or `value` (case-insensitive).
- **Keyboard** → `↑/↓` wrap, `Enter` selects, `Esc` closes.
- **Outside click** → closes (capture-phase listener); clicks inside `#enum-picker-react` are ignored.

## Atom wiring

| Atom | Role |
|------|------|
| `enumPickerStateAtom` | `null` ⇒ closed. `{ anchor: { stepId, fieldName }, triggerRect }` ⇒ open over that field |
| `commandsAtom` | Source of `fields[fieldName].options` and `default` for initial highlight |
| `stepsAtom` | Provides `command` + `params` for initial highlight |
| `setParamAtom` | Persists the selected value into step state |
